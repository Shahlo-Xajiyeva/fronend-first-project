# fronend-first-project
# 💻 Shaxsiy Portfolio Sahifasi (Personal Portfolio)

Bu mening dasturlash olamidagi ilk qadamlarimdan biri bo'lgan shaxsiy vizitka va portfolio sahifam. Loyihada o'zim haqimdagi ma'lumotlar, qiziqishlarim va texnologik ko'nikmalarim aks etgan.

---

## 🚀 Loyiha Haqida

Ushbu kichik veb-sahifa butunlay **Frontend** texnologiyalari yordamida yaratilgan. Maqsad — vizual jihatdan zamonaviy, tez yuklanadigan va minimalistik dizaynga ega shaxsiy sahifa tayyorlash edi.

### 🛠 Ishlatilgan Texnologiyalar:
* **HTML5** — Sahifaning tuzilishi va ma'lumotlar tartibi uchun.
* **CSS3** — Ranglar, joylashuv (Flexbox), soyalar va animatsiyalar bilan ishlash uchun.

---

## 🎨 Loyiha Imkoniyatlari (Features)
* ✨ Zamonaviy va toza vizual dizayn.
* 📱 Barcha qurilmalar (mobil telefonlar, planshetlar va kompyuterlar) uchun moslashuvchan interfeys (Responsive design).
* ⚡️ Hover animatsiyalar (sichqoncha bilan ustiga kelganda blokning biroz ko'tarilishi).
